async4j

Callback based asynchronous programming library to harness hardware performances

No more callbacks hell

One of design principle of async4j is no application logic in callbacks, so more asynchronous clean code. Callbacks are used to capture asynchronous operation completion or execption.

Asynchronous controls

Asynchronous pipe, condition, while, foreach and the most noticeable one the asynchronous parallel foreach with flow control. In fact all of well known control in synchronous contructs can be translated to thier asynchronous equivalent openning the dore for larger asynchronous algorithms design.

Java support

Shared callback implementations as core library, DSL to write controls specifically to the language exprissiveness. Builtin DSL for Java 5 that fit well in Java 8 by the concept of functional interfaces. A DSL implemented for scala that take advantage of delimited continuation and for comprehension.
