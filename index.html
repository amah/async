async4j

Callback based asynchronous programing library

Welcome

Async4j is a library that provides callback based building blocks for asynchronous programing model: pipe, try/catch/finaly, condition, foreach and even parallel foreach.

At the heart of async4j library is the Callback interface used to capture the completion of asynchronous call, whatever normal or abnormal completion: 


// code here
That interface has nothing very new, it is the well known completion event listener widely used in existing asynchronous interaction where the callback implements the logic to be run when the asynchronous call ends. The difference with asybc4j is that the callback interface is not intended to implement the business logic on completion but the asynchronous flow logic.
